# Прикладное программирование
## Название
Лабораторная 4
## Описание
Программа для нахождение оптимальной рассадки людей в такси, для максимальной экономии при поездках.
## Рекомендации
Язык написания python версии 3.9. Рекомендуемая среда - Visual studio code
